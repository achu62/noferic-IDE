# BUILD  FROM SOURCE<br>
## Dependencies<br>
Node(LTS recommended)<br>
npm(comes with node)<br>
git(optional,if the zip is downloaded)<br>
### run commands<br>
```bash
mkdir noferic-ide
cd noferic-ide
gh repo clone achu62/noferic-IDE
```
(you can also download the .zip)
```bash
npm install

```

## on linux:

```bash
npm run build -- --linux

```

## on Windows
```bash
npm run build -- --win
```
the executable or appimage can be found in appoutput folder
